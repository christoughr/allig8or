# allig8tor — 내가 할 일 (천천히, 한 단계씩)

코드·배포는 이미 **allig8tor** 브랜드로 맞춰 두었어요.  
아래는 **당신이 브라우저에서** 직접 해야 하는 것만 정리했습니다.  
한 번에 다 하지 말고, **체크 하나 끝낼 때마다** 다음으로 가세요.

## 0단계 — 지금 상태 확인 (2분)

브라우저에서 열어보기:

| 확인 | URL |
|------|-----|
| 메인 | https://www.allig8tor.com |

로고·푸터에 **allig8tor** 로 보이면 브랜드 변경 반영된 것.

## 1단계 — Supabase 로그인 URL (필수, 약 5분)

**왜:** 회원가입·로그인·이메일 확인이 새 도메인에서만 동작함.

1. https://supabase.com/dashboard 로그인  
2. allig8or 프로젝트 선택  
3. 왼쪽 **Authentication** → **URL Configuration**  
4. 아래처럼 입력 후 **Save**

| 항목 | 값 |
|------|-----|
| **Site URL** | `https://www.allig8tor.com` |
| **Redirect URLs** (한 줄씩 추가) | `https://www.allig8tor.com/auth/callback` |
| | `https://www.allig8tor.com/**` |
| | `https://allig8tor.com/**` |
| | `https://allig8or.com/**` ← 예전 도메인 (몇 주만 유지) |

5. **테스트:** https://www.allig8tor.com/login → 가입 또는 로그인 → `/app` 으로 들어가지면 성공.

---

## 2단계 — Vercel 도메인 (대부분 완료, 확인만)

**왜:** 예전 주소가 새 주소로 영구 이동해야 SEO·북마크가 안 깨짐.

1. https://vercel.com/onlyus/allig8or/settings/domains  
2. 확인:
   - `allig8tor.com`, `www.allig8tor.com` → **Valid**
   - `allig8or.com`, `www.allig8or.com` → **Redirect** → `allig8tor.com` (또는 www)
3. 리다이렉트 종류: **308 Permanent** (307/302 아님)

**로컬 PC에서 Vercel 연결이 안 될 때** (한 줄씩 실행):

```powershell
cd $env:TEMP\allig8or
vercel login
vercel link --yes --project allig8or --scope onlyus
```

> GitHub·Vercel **프로젝트 이름**은 `allig8or` 로 남아 있어도 됩니다. **사이트에 보이는 이름**만 allig8tor 입니다.

---

## 3단계 — Google 검색 등록 (SEO, 약 15분)

**왜:** 구글에 사이트맵을 알려야 검색 결과에 올라갈 수 있음. (순위 1페이지는 시간이 더 걸림)

1. https://search.google.com/search-console  
2. **속성 추가** → URL: `https://www.allig8tor.com`  
3. 소유권 확인: **HTML 태그** 방식 선택 → `content="..."` 안의 코드만 복사  
4. Vercel → https://vercel.com/onlyus/allig8or/settings/environment-variables  
   - 이름: `NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION`  
   - 값: 복사한 코드  
   - Environment: **Production**  
5. 저장 후 **Redeploy** (Deployments → 최신 배포 → ⋯ → Redeploy)  
6. Search Console에서 **확인** 클릭  
7. **Sitemaps** → 새 사이트맵 추가:  
   `https://www.allig8tor.com/sitemap.xml`  
8. **URL 검사** → `https://www.allig8tor.com` → **색인 생성 요청**

## 5단계 — Lemon Squeezy (스토어 승인 **후**)

**지금은 안 해도 됨.** 스토어가 아직 승인 안 됐으면 Vercel에 결제 키가 없는 게 정상입니다.

승인 메일 받은 **그때**:

1. Lemon Squeezy → API 키, Webhook 시크릿, Variant ID 복사  
2. Vercel Environment Variables에 추가:
   - `LEMONSQUEEZY_API_KEY`
   - `LEMONSQUEEZY_WEBHOOK_SECRET`
   - `NEXT_PUBLIC_LEMONSQUEEZY_STORE_ID`
   - `NEXT_PUBLIC_LEMONSQUEEZY_VARIANT_PRO`
   - `NEXT_PUBLIC_LEMONSQUEEZY_VARIANT_TEAM`
3. Webhook URL (반드시 **www**):
   `https://www.allig8tor.com/api/webhooks/lemonsqueezy`
4. Redeploy  
5. 브라우저에서 확인:  
   https://www.allig8tor.com/api/webhooks/lemonsqueezy  
   → `"billing":"configured"` 나오면 OK
