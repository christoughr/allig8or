# 사이트맵 다시 제출 (Google + Bing)

라이브 사이트맵: **https://www.allig8tor.com/sitemap.xml** (현재 **13개** URL)

---

## Google Search Console

1. https://search.google.com/search-console  
2. 속성: `https://www.allig8tor.com`  
3. 왼쪽 **Sitemaps** (사이트맵)  
4. **기존 `/sitemap.xml` 행이 있으면**
   - Status가 Success이고 Discovered pages가 **13** 근처면 → **추가 작업 없음**
   - 예전 **4**에서 안 바뀌면 → 행 오른쪽 **⋮** → **Delete sitemap**  
5. **Add a new sitemap** / **새 사이트맵 추가**  
6. 입력 (둘 중 하나, 보통 짧은 쪽):

```text
sitemap.xml
```

7. **Submit** / **제출**  
8. 몇 분~24시간 후 **Discovered pages: 13** 확인

---

## Bing Webmaster Tools

1. https://www.bing.com/webmasters  
2. 사이트 `https://www.allig8tor.com` 선택  
3. **Sitemaps**  
4. 제출 (전체 URL 또는):

```text
https://www.allig8tor.com/sitemap.xml
```

또는

```text
sitemap.xml
```

5. Status **Success** / Submitted 확인

---

## 새 글 push 후

1. `content/blog/새글.md` 추가 → `git push`  
2. 위와 같이 Google·Bing에서 사이트맵 **다시 제출** (또는 기다리면 자동 재크롤)  
3. (선택) https://www.allig8tor.com/api/indexnow 브라우저에서 열기
