# 새 블로그 글 추가하기 (allig8tor)

블로그 글은 **Markdown 파일** 하나 추가 → **Git push** → Vercel이 자동 배포합니다.  
코드 수정 없이 `content/blog/` 폴더만 건드리면 됩니다.

---

## 1. 파일 만들기

경로:

```text
content/blog/여기에-slug.md
```

- **slug** = URL에 쓰이는 이름 (영문, 하이픈).  
  예: `my-new-post.md` → `https://www.allig8tor.com/blog/my-new-post`

---

## 2. 파일 맨 위에 frontmatter (필수)

아래 블록을 **파일 첫 줄**에 붙여 넣고 값만 바꿉니다.

```markdown
---
title: 글 제목 (검색 결과에 보임)
description: 한 줄 요약 (검색·RSS에 보임, 150자 내외 권장)
date: 2026-05-27
keywords:
  - 키워드1
  - 키워드2
---

# 본문 첫 제목

여기부터 일반 마크다운으로 작성...
```

| 필드 | 필수 | 설명 |
|------|------|------|
| `title` | ✅ | 페이지 제목 |
| `description` | ✅ | SEO 설명 |
| `date` | ✅ | `YYYY-MM-DD` |
| `keywords` | 선택 | SEO 키워드 목록 |

---

## 3. 본문 작성 (마크다운)

지원 예시:

```markdown
## 소제목

일반 문단.

- 불릿 목록
- 두 번째 항목

**굵게** 강조.

내부 링크: [앱 열기](/app?tool=presentation)

외부 링크: [Google](https://google.com)
```

글 끝에 CTA 권장:

```markdown
[지금 생성하기 →](/app?tool=presentation)
```

`tool` 값: `website` | `presentation` | `spreadsheet` | `document` | `pdf`

## 5. Git commit & push

로컬에서 repo 루트(`allig8or` 폴더)에서:

```powershell
cd C:\path\to\allig8or

git add content/blog/startup-budget-template-ai.md
git commit -m "blog: add startup budget template guide"
git push origin main
```

- `main`에 push하면 Vercel이 **자동으로 Production 배포** (~1–2분).
- 배포 확인: https://vercel.com/onlyus/allig8or/deployments

---

## 6. 배포 후 (SEO, 사무실 PC에서 해도 됨)

1. 글이 보이는지: `https://www.allig8tor.com/blog/여기-slug`  
2. **Google Search Console** → **URL 검사** → 위 URL 입력 → **색인 생성 요청**  
3. (선택) 사이트맵은 자동 갱신됨. 필요하면 Sitemaps에서 `sitemap.xml` 다시 제출.

새 글 여러 개 push 후 IndexNow (선택):

```text
https://www.allig8tor.com/api/indexnow
```

브라우저에서 열거나 `curl` — Bing 등에 URL 알림.

---

## 7. 유스케이스 페이지 추가 (고급)

블로그와 별도로 랜딩형 페이지는 `lib/use-cases.ts`의 `USE_CASES` 배열에 항목 추가 + `app/use-cases/[slug]`가 자동 생성됩니다.  
일반 글만 쓸 때는 **이 파일 건드릴 필요 없음**.

---

## 문제 해결

| 증상 | 확인 |
|------|------|
| 404 | slug와 파일명이 같은지 (`foo.md` → `/blog/foo`) |
| 빌드 실패 | frontmatter `---` 위아래 줄바꿈, `date` 형식 |
| 목록에 안 보임 | `date`가 너무 미래가 아닌지, push 했는지 |
