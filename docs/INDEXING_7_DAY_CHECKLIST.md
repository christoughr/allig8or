# 7-Day Indexing Checklist

Follow this once per day for the next 7 days.

## Daily (5-10 minutes)

1. Open Google Search Console and inspect:
   - `https://www.allig8tor.com/`
   - `https://www.allig8tor.com/blog`
   - One newly published blog URL
2. If URL is not indexed yet, click **Request indexing**.
3. Open Bing Webmaster Tools and inspect the same 3 URLs.
4. Submit indexing request for URLs still not indexed.
5. Check `https://www.allig8tor.com/sitemap.xml` loads with HTTP 200.
6. Publish at least 1 new content page every 2-3 days.

## Every 2-3 days

1. Add 2 internal links from old pages to newest blog post.
2. Add 1 internal link from newest post to a use-case page.
3. Share newest post URL on one external channel (social/profile/community).

## Day 7 review

1. In GSC Performance, compare:
   - Total impressions
   - Average position
   - Top queries and top pages
2. Keep expanding content around queries that already have impressions.
